## 1?? GOOGLE SEARCH CONSOLE (CRITICAL)

### Step 1: Create Google Account
```
If you don't have one, visit: https://accounts.google.com/signup
```

### Step 2: Add Property to Google Search Console

**URL**: https://search.google.com/search-console/about

**Steps**:
```
1. Click "Start now"
2. Sign in with your Google account
3. Choose "URL prefix" property type
4. Enter your site: https://mcbdshost.com
5. Click "Continue"
```

### Step 3: Verify Site Ownership

**Choose ONE verification method** (Google Analytics is easiest if already installed):

**Option A: Google Analytics** (Easiest - Already Installed)
```
1. Google Search Console will detect your Google Analytics
2. Click "Verify" next to Google Analytics
3. Done! (Instant verification)
```

**Option B: HTML File** (Alternative)
```
1. Download the HTML verification file
2. Upload to: https://mcbdshost.com/[filename].html
3. Click "Verify"
```

**Option C: HTML Meta Tag** (Alternative)
```
1. Copy the meta tag provided
2. Add to <head> of your main page (MainLayout.razor)
3. Click "Verify"
```

**Option D: DNS Record** (Technical)
```
1. Add TXT record to your DNS settings
2. Wait for DNS propagation (24-48 hours)
3. Click "Verify"
```

### Step 4: Submit Sitemap

```
After verification:

1. Click "Sitemaps" in left menu
2. Click "Add/test sitemap"
3. Enter: https://mcbdshost.com/sitemap.xml
4. Click "Submit"

Watch status:
- Submitted ?
- Indexed: 6 pages (should increase to 6)
- Errors: 0 (should remain 0)
```

### Step 5: Monitor Performance

```
1. Go to "Performance" tab
2. Monitor:
   - Total impressions (search appearance)
   - Click-through rate
   - Average position
   - Top queries
3. Monitor "Coverage" tab:
   - Valid pages (should be 6)
   - Excluded pages
   - Errors
```

### Step 6: Request Indexing

```
For faster indexing:

1. Click "Inspect URL" (top bar)
2. Enter: https://mcbdshost.com/
3. Click "Test live URL"
4. Click "Request indexing" (if available)
5. Repeat for each page:
   - /features
   - /get-started
   - /contact
```

### Performance Metrics

1. **Valid Pages**: Should be 6
2. **Errors**: Should be 0
3. **Excluded**: Should be 0
4. **Pending**: New pages

### Engagement Metrics
1. **Organic traffic**: Should increase
2. **Time on site**: Minutes spent
3. **Bounce rate**: Lower is better (<70%)
4. **Conversions**: Contact forms, downloads

## ?? Common Issues & Solutions

### Issue: "Noindex" Tag
**Problem**: Pages not indexing
**Solution**: Remove noindex meta tag (if present)
```html
<!-- Remove this if present -->
<meta name="robots" content="noindex">
```

### Issue: Crawl Errors
**Problem**: Google can't access pages
**Solution**: 
- Check robots.txt (shouldn't block site)
- Verify HTTPS works
- Check for 404 errors

### Issue: Slow Indexing
**Problem**: Pages not indexing quickly
**Solution**:
- Use URL inspection tool
- Request indexing manually
- Build quality backlinks
- Share on social media

### Issue: 404 Errors
**Problem**: Removed pages still indexed
**Solution**:
- Set proper 404 redirects
- Create 404 page
- Use Google Search Console to remove
