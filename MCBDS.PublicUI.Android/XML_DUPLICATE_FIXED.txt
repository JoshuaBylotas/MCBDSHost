=== Android XML Configuration Error - FIXED ===

ERROR MESSAGE:
"Caused by: android.security.net.config.XmlConfigSource$ParserException: 
winserver03.bylotas.net has already been specified"

ROOT CAUSE:
Duplicate domain entries in network_security_config.xml

THE PROBLEM:
```xml
<domain includeSubdomains="true">bylotas.net</domain>
<domain includeSubdomains="true">winserver03.bylotas.net</domain>  <!-- DUPLICATE! -->
<domain includeSubdomains="true">WINSERVER03.bylotas.net</domain>  <!-- DUPLICATE! -->
```

When you specify "bylotas.net" with includeSubdomains="true", it ALREADY includes:
- winserver03.bylotas.net
- WINSERVER03.bylotas.net
- mail.bylotas.net
- any.subdomain.bylotas.net

Adding them again causes Android to reject the configuration file!

THE FIX:
```xml
<!-- This is all you need -->
<domain includeSubdomains="true">bylotas.net</domain>
```

WHAT includeSubdomains="true" MEANS:
? bylotas.net
? winserver03.bylotas.net
? www.bylotas.net
? ANY.SUBDOMAIN.bylotas.net
