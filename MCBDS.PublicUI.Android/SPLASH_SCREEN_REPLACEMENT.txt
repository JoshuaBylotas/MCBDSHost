=== Splash Screen Replacement Guide ===

WHAT WAS CHANGED:
Updated MCBDS.PublicUI.Android to use custom "MCBDS Splash.png" instead of default splash.svg

FILE MODIFIED:
? MCBDS.PublicUI.Android\MCBDS.PublicUI.Android.csproj

WHAT YOU NEED TO DO:

STEP 1: Add Your Image File
----------------------------
1. Locate your "MCBDS Splash.png" file
2. Copy it to this location:
   MCBDS.PublicUI.Android\Resources\Splash\
3. Rename it to: mcbds_splash.png (remove spaces, lowercase)
   
   Final path should be:
   MCBDS.PublicUI.Android\Resources\Splash\mcbds_splash.png

STEP 2: Verify Configuration
----------------------------
The .csproj file has been updated with:
  <MauiSplashScreen Include="Resources\Splash\mcbds_splash.png" Color="#FFFFFF" BaseSize="456,456" />

- Color="#FFFFFF" = White background (change if needed)
- BaseSize="456,456" = Recommended size for square images

STEP 3: Adjust Settings (Optional)
----------------------------
If your splash image needs different settings, edit the .csproj line:

Background Color Options:
- Color="#FFFFFF" - White background
- Color="#000000" - Black background  
- Color="#512BD4" - Purple (original default)
- Color="#1E1E1E" - Dark gray
- Color="Transparent" - Transparent background

Image Size Options:
- BaseSize="456,456" - Square image (recommended)
- BaseSize="512,512" - Larger square
- BaseSize="1024,768" - Landscape
- BaseSize="768,1024" - Portrait

STEP 4: Build and Test
----------------------------
1. Clean solution
2. Rebuild MCBDS.PublicUI.Android
3. Deploy to emulator/device
4. Launch app - you should see your custom splash screen!

IMAGE REQUIREMENTS:
----------------------------
? Format: PNG (with transparency if desired)
? Recommended size: 1024x1024 or 2048x2048 pixels
? Should work on various screen sizes/densities
? Centered automatically by MAUI
? Background color fills screen around image

TROUBLESHOOTING:
----------------------------
If splash doesn't appear:
1. Verify file exists at: Resources\Splash\mcbds_splash.png
2. Check file is included in project (right-click ? Properties ? Build Action)
3. Clean and rebuild
4. If file has spaces in name, remove them
5. Ensure file extension is lowercase (.png not .PNG)
