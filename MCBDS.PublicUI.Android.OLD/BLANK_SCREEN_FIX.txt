# Quick Fix for Blank Screen Issue

## Problem
The app shows splash screen, then goes blank. This means:
1. ? MAUI app launched successfully
2. ? MainActivity started
3. ? BlazorWebView failed to load/render

## Immediate Checks

### 1. View Android Logs NOW
```bash
# Open a terminal and run:
adb logcat | findstr "BlazorWebView\|Blazor\|Exception\|Error\|chromium"
```

Look for errors related to:
- BlazorWebView initialization
- Missing index.html
- JavaScript errors
- Component loading failures

### 2. Quick File Check
Verify these files exist:

```
MCBDS.PublicUI.Android/
??? wwwroot/
?   ??? index.html ?
?   ??? app.css ?
?   ??? lib/
?       ??? bootstrap/ ?
??? Components/
?   ??? Routes.razor ?
?   ??? _Imports.razor ?
?   ??? Layout/
?   ?   ??? MainLayout.razor ?
?   ?   ??? NavMenu.razor ?
?   ??? Pages/
?       ??? Home.razor ?
```

### 3. Check MainPage.xaml

The BlazorWebView must point to correct component:
```xml
<RootComponent Selector="#app" ComponentType="{x:Type components:Routes}" />
```

## Most Likely Causes

### Cause 1: Missing Routes Component
**Check**: Does `Components/Routes.razor` exist and reference correct Layout?

### Cause 2: Missing index.html or Wrong Content
**Check**: Does `wwwroot/index.html` have `<div id="app">`?

### Cause 3: Component Loading Failed
**Check**: Android Logcat for Blazor initialization errors

### Cause 4: Assets Not Deployed
**Fix**: Clean and rebuild
```bash
cd MCBDS.PublicUI.Android
rm -r bin obj
dotnet build -f net10.0-android
```
